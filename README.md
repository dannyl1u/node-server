# Node API 

## Setup

1. Clone this repository

   ```bash
   $ git clone https://github.com/dannyl1u/node-api.git
   ```


2. Navigate into the project directory

   ```bash
   $ cd node-api
   ```

3. run npm install

   ```bash
   $ npm install
   ```

4. Run the server

   ```bash
   $ node .
   ```
   
Server should now be up at [http://localhost:8080](http://localhost:8080)!
